## 「 Установка приложения 」

### ➢ Предварительные требования

Перед установкой приложения убедитесь, что на вашем компьютере установлены следующие компоненты:

#### 1. .NET 9.0 Runtime
**Необходимо для работы WPF приложения**

https://dotnet.microsoft.com/en-us/download/dotnet/9.0

#### 2. WebView2 Runtime  
**Необходимо для корректного отображения функций**

https://developer.microsoft.com/en-us/microsoft-edge/webview2/?form=MA13LH

### ➢ Процесс установки

#### Шаг 1: Скачайте архив приложения
- Скачайте архив `AlphaBankMobileOfficeApp-win-x64-fdd.zip`

#### Шаг 2: Распакуйте архив
- Создайте папку `AlphaBank Mobile Office` в удобном месте
- Распакуйте все файлы из архива в созданную папку

#### Шаг 3: Запустите приложение
- Запустите файл `AlphaBankMobileOfficeApp.exe`

### ➢ Вызов мобильного офиса

Интерактивная система заказа банковского автомобиля с выбором адреса и времени через карты Яндекс.

   Ключевые возможности:
-  **Точное позиционирование** - интегрированные карты с геокодированием
-  **Гибкое расписание** - выбор удобного времени
-  **Мгновенное подтверждение** - быстрая обработка запроса

 Технологии:
- Yandex Maps API для точного позиционирования
- WebView2 для интеграции карт
- Геокодирование адресов в координаты
- Асинхронные операции для плавного UI
